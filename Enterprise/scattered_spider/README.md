# Scattered Spider

[![License](https://img.shields.io/badge/License-Apache_2.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)

Based on open-source intelligence, the MITRE ATT&CK&reg; Evaluations team
created the Scattered Spider scenario leveraging techniques seen in operations
in the wild. The scenario was designed based on tools, resources, and
intelligence available at the time.
