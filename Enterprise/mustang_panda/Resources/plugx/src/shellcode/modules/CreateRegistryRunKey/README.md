## CreateRegistryRunKey Module

This module creates a run-key in the registry for persistence. This module is specific to the Mustang Panda scenario.
The run-key created is for an executable named `WinGupSvc.exe`.

The run-key path is: `C:\Users\<current user>\AppData\Local\Temp\WinGupSvc.exe`

### Default Dispatch Code

`0x1002`

### Required Arguments

No arguments are required.
